# TensorFlow

## Primeiro caso
Considere que desejamos construir uma fórmula capaz de calcular o preço de passagens aéreas em função do combustível despendido durante cada voo. A única informação que temos é uma massa de dados contendo o valor e a quantidade de combustível utilizada em cada viagem.

## Iniciando a virtualenv
```sh
virtualenv -p python3 venv
```

## Executando o ambiente virtual
```sh
. venv/bin/activate
```

## Instalando o tensorflow
```sh
pip3 install tensorflow
```
